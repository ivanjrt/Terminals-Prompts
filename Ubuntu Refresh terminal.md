# Installing prerequisites:
```
sudo apt-get update && sudo apt-get upgrade
sudo apt-get install -y git vim
```

# Cloning the repo:
```
git clone https://github.com/pixegami/terminal-profile.git
cd terminal-profile
./install_powerline.sh
./install_terminal.sh
```
